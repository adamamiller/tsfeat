In this website, we present the FATS (Feature Analysis for Time Series) library. FATS is a Python library which facilitates and standardizes feature extraction for time series data. In particular, we focus on one application: feature extraction for astronomical light curve data, although the library is generalizable for other uses. We detail the methods and features implemented for light curve analysis, and present examples for its usage.

For a deeper understanding of FATS the user can visit the library website, arXiv article or the code documents.

It is also possible to obtain the library by downloading the python package from https://pypi.python.org/pypi/FATS or by directly installing it from the terminal as follows:

$ pip install FATS

How to cite FATS:

  • make reference to the arXiv article
  • directly use the Bibtex entry
  • copy the following formatted citation: \bibitem[Nun et al.(2015)]{2015arXiv150600010N} Nun, I., Protopapas, P., Sim, B., et al.\ 2015, arXiv:1506.00010
